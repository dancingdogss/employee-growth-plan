📦 Bulk replenish (create many)
▼
How bulk replenish works:
Select a stock type, starting number, and how many units to create.
Example: Type B, start 1, quantity 4 → creates B-0001, B-0002, B-0003, B-0004.
Each B unit = 10 physical items (1 sale event = 100 earnings).
Each M unit = 5 physical items (1 sale event = 100 earnings).
Each C unit = 1 physical item (1 sale event = 100 earnings).
Existing stock IDs are skipped — no duplicates created.
